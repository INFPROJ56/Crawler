Java engineer E-commerce

Kan jij als Java engineer de ontwikkeling van complexe e-commerce sites verbeteren?

Als Java engineer  werk je in de rol van backend developer met Demandware, een specialistisch e-commerce software platform dat gebruikt word door retailers over de hele wereld. (Denk aan bedrijven als Quicksilver, UGG, Panasonic, G-Star, Suit Supply, Playmobil, Adidas etc…)

In je nieuwe rol blink je met je team uit in de ontwikkeling van complexe e-commerce websites die vragen om een hoge kwaliteitcode-oplossingen zowel in de uitvoering,verbetering alsondersteuningvane-commerce platforms.

Denk daarbij aan oplossingen voor online commerce websites en daarbij behorende systemen zoals Payment system integratie, Order Management integratie en Content Management integratie. Daarnaast draag je er uiteraard zorg voor dat het project aan de hand van nauwkeurige inschatting, voor de deadline opgeleverd kan worden.

Vereisten:

Diepgaande passie voor e-commerce.

5 jaar ervaring met webservices, XML, HTML, CSS, Ajax, Java, Javascript ( jQuery, JSON ).

Ontwikkel ervaring met Demandware of Intershop.

Wat staat er tegenover?

Geweldige klanten om voor te werken.

Jonge, flexibele cultuur met zeer creatieve mensen.

Uitstekend salaris.

Wil jij volledig geïntegreerde oplossingen bouwen voor bekende internationale bedrijven?
